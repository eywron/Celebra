Celebra

About Celebra

Celebra is a lightweight, experimental chat interface demonstrating generative AI capabilities with a focus on a clean, dark neon aesthetic and fast interaction.

  • Context-aware conversation flow with local history support.
  • Text generation powered by backend-hosted generative models.
  • Secure: API keys remain server-side via a serverless proxy (recommended deployment: Vercel).

Privacy: conversation history is stored locally in your browser by default; no personal data is sent to third parties by the demo itself. For production use, configure server-side storage and appropriate data handling policies.

Source & issues: github.com/eywron/Celebra
